# Orders App

Aplicação mobile em React Native com Expo para gerenciamento de pedidos.
O backend da aplicação se encontra no seguinte [repositório](https://github.com/Guscaleman/mobile-code-challenge)

## Tecnologias
- React Native
- Expo
- Axios
- React Native Paper

## Como rodar
```
npm install
```
em seguida:
```
npm expo start
```

## Como testar no seu celular

Baixe o aplicativo Expo Go na sua loja de aplicativos, aponte a câmera do seu celular no QR Code que é gerado no terminal após rodar sua aplicação localmente.
